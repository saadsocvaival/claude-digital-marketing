# Archive

Deprecated specs live here. Policy: never delete. When a spec is superseded, move it to `99-archive/<original-relative-path>`, set frontmatter `status: deprecated`, and add `superseded_by: <new-path>`. The execution ledger may still reference historical specs by their archived path, so archival must preserve path structure.
