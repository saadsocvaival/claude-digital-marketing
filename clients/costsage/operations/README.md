# CostSage — Marketing Operations

This directory is the **single source of truth** for cross-vertical Digital Marketing execution at CostSage.

## Read in this order

1. **`DIGITAL-MARKETING-DEPT-OPERATIONAL-PLAN.md`** — the master plan. Covers all 8 verticals (SEO/AEO/GEO, Web/CRO, Paid Media, Content, Social, Email, Analytics, Brand) with consistent structure: outcomes → workflows → tools/accounts → executable-now → blocked → KPIs → tracking artifact.
2. **`STATUS-TRACKER.md`** — the status board. One row per vertical. Updated every Monday + on material events. Stakeholder-readable in 30 seconds.

## Sub-directories (created as work begins)

- `campaigns/` — V3 paid-media campaigns, one folder per active campaign (brief, weekly report, post-mortem)
- `content/` — V4 editorial calendar, content briefs, repurposing rules
- `social/` — V5 posting queue, engagement log, weekly reports
- `email/` — V6 sequences, newsletter issues, lifecycle map, deliverability log
- `analytics/` — V7 tag plan, UTM convention, KPI dictionary, dashboard spec
- `brand/` — V8 voice guidelines, asset library, category-design, narrative-thesis
- `web-cro/` — V2 CRO test history (one folder per A/B test)
- `snapshots/` — monthly top-line snapshots for trend analysis
